# 间距 & 圆角 & 阴影

## 间距刻度

| Token | 值 | 用途 |
|---|---|---|
| `--sp-1`  | 4px  | 行内最小 gap |
| `--sp-2`  | 8px  | 标签内、icon 与文字 |
| `--sp-3`  | 12px | 表单元素 |
| `--sp-4`  | 16px | 一般元素之间 |
| `--sp-5`  | 20px | 卡片内边距 |
| `--sp-6`  | 24px | 区块之间 |
| `--sp-8`  | 32px | 页面分组 |
| `--sp-10` | 40px | 顶级 section 之间 |

## 圆角

| Token | 值 | 用途 |
|---|---|---|
| `--radius-sm` | 8px  | 小标签、chip |
| `--radius-md` | 12px | 输入框、按钮 |
| `--radius-lg` | 16px | 卡片（**主要圆角**） |
| `--radius-xl` | 20px | 弹窗、大容器 |

> 整体偏**大圆角**，凸显柔和风格。小到 4px 的尖锐圆角不要出现。

## 阴影

| Token | 值 | 用途 |
|---|---|---|
| `--shadow-soft` | `0 6px 24px -8px rgba(78, 118, 170, 0.18)` | 卡片常态 |
| `--shadow-pop`  | `0 10px 30px -10px rgba(78, 118, 170, 0.35)` | hover / 弹窗 |

阴影颜色**带蓝调**，避免灰死；不要用纯黑阴影。

## 栅格

- 主内容区最大宽度：**1280px**（超大屏居中）
- 内容 grid：画页库 `2/3/4/5 列自适应`，本管理 `1/2/3`
- 统计卡片：`col-span-2` + 2x2 图表网格
