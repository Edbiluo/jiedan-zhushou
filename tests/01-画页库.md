# 01 · 画页库

## TC-P-01 上传一张普通画页
- **类型**: 正向
- **前置**: 款式「特价款」、尺寸「中号」已存在
- **步骤**: 1. 点击「上传画页」 2. 选择一张本地 JPG 图 3. 标题填"樱花涂鸦"，款式=特价款，尺寸=中号，时长=1.5h 4. 点击上传
- **预期**: 画页库列表出现新画页，缩略图可见；标题/款式/尺寸/时长正确显示

## TC-P-02 上传一张封页
- **类型**: 正向
- **前置**: 无
- **步骤**: 1. 点击上传 2. 勾选"这是封页" 3. 款式/尺寸字段应隐藏或禁用 4. 填标题、时长=2h，提交
- **预期**: 画页带「封页」标签，无款式/尺寸；列表中可筛选"只看封页"定位到

## TC-P-03 原图较大时的响应
- **类型**: 正向
- **前置**: 准备一张 10MB 的图
- **步骤**: 上传同 TC-P-01
- **预期**: 上传成功；列表展示的是缩略图（加载迅速）；点击详情可看到原图

## TC-P-04 按款式筛选
- **类型**: 正向
- **前置**: 不同款式各有 2+ 画页
- **步骤**: 款式下拉选择「重工款」
- **预期**: 列表仅展示重工款画页；再选「全部款式」恢复所有

## TC-P-05 按封页筛选
- **类型**: 正向
- **前置**: 同时存在封页和普通画页
- **步骤**: 勾选「只看封页」
- **预期**: 列表仅显示封页

## TC-P-06 搜索标题
- **类型**: 正向
- **前置**: 存在标题含"樱花"的画页
- **步骤**: 搜索框输入"樱花"回车
- **预期**: 只显示标题匹配的画页；清空搜索恢复

## TC-P-07 编辑画页
- **类型**: 正向
- **前置**: 任一画页存在
- **步骤**: hover 画页点编辑按钮，修改标题和时长，保存
- **预期**: 画页信息更新，排期若受影响要自动跟进（见 03-排期）

## TC-P-08 删除画页
- **类型**: 正向
- **前置**: 画页未被任何本引用
- **步骤**: 点删除按钮 → 确认
- **预期**: 画页从列表消失；对应缩略图/原图文件可被清理

## TC-P-09 空态
- **类型**: 边界
- **前置**: 无画页
- **步骤**: 打开画页库
- **预期**: 显示空态文案（"还没有画页..."），有引导按钮

## TC-P-10 非法时长
- **类型**: 异常
- **前置**: 无
- **步骤**: 上传时时长填 0 或 -1
- **预期**: 拒绝保存或回退默认值（至少 0.25h）

## TC-P-11 不支持的图片格式
- **类型**: 异常
- **前置**: 准备一个 .tiff 或损坏的图
- **步骤**: 尝试上传
- **预期**: 提示格式不支持或读取失败，不入库

## TC-P-12 款式被删除后存量画页的表现
- **类型**: 边界
- **前置**: 创建自定义款式"粉彩款"并上传一张画页，后删除该款式（见 07-设置）
- **预期**: 画页仍可见，款式字段显示空 / "未分类"，不影响其他操作
