# 02 · 本管理

## TC-B-01 创建本（选已有画页）
- **类型**: 正向
- **前置**: 画页库已有 3+ 画页
- **步骤**: 1. 点"新建本" 2. 填本名"小鹿的本"，单价 500，交付 14 天后 3. 多选 3 张画页 4. 提交
- **预期**: 本列表出现新本，状态=进行中；排期自动生成（见 03-排期）

## TC-B-02 创建本时上传新画页
- **类型**: 正向（如果 UI 支持）
- **前置**: 无
- **步骤**: 新建本弹窗中选择"上传新画页"并添加到本
- **预期**: 画页入库 + 关联到本

## TC-B-03 状态：进行中
- **类型**: 正向
- **前置**: 本 deadline 距离今天 > 3 天，画页未全完成
- **预期**: 状态自动为"进行中"（chip 蓝色）

## TC-B-04 状态：快到交付
- **类型**: 正向
- **前置**: 本 deadline 距离今天 ≤ 3 天，未完成
- **预期**: 状态自动切为"快到交付"（chip 橙色）

## TC-B-05 状态：延期
- **类型**: 边界
- **前置**: deadline 已过但本未完成
- **预期**: 状态自动切为"延期"（chip 红色），日历上相关项高亮警示

## TC-B-06 状态：已完成
- **类型**: 正向
- **前置**: 本所有画页都标记完成（或点"标记整本完成"）
- **预期**: 状态=已完成（chip 绿色），撒花动效触发一次

## TC-B-07 改交付日期触发重排
- **类型**: 正向
- **前置**: 本已有排期
- **步骤**: 把 deadline 改得更近（或更远）
- **预期**: 剩余未完成的画页按新 deadline 重新排布（用户手动拖拽过的保留）

## TC-B-08 状态筛选
- **类型**: 正向
- **前置**: 不同状态的本都存在
- **步骤**: 切换顶部状态筛选
- **预期**: 只看对应状态的本

## TC-B-09 删除本
- **类型**: 正向
- **前置**: 任意一本
- **步骤**: 在本详情点击"删除本" → 确认
- **预期**: 本从列表消失，相关排期/关联自动清理；画页本身保留

## TC-B-10 单价为 0
- **类型**: 边界
- **前置**: 无
- **步骤**: 创建本时单价填 0
- **预期**: 允许保存（如朋友单），统计时按 0 计入

## TC-B-11 画页数 0
- **类型**: 异常
- **前置**: 不选任何画页
- **步骤**: 尝试提交
- **预期**: 提示"至少选择一个画页"，不创建

## TC-B-12 deadline 设为今天或历史
- **类型**: 异常
- **前置**: 无
- **步骤**: deadline 选为今天 / 昨天
- **预期**: 允许保存（不强制校验），但状态会立刻落入"快到交付"或"延期"

## TC-B-13 批量创建多本并行排期
- **类型**: 边界
- **前置**: 同一天内创建 3 本 deadline 相近的本
- **预期**: 排期算法能多本并行分配一天的工时，不冲突
